# London Property Investment Intelligence System

## Executive Summary

This project aims to develop a data-driven Decision Intelligence System that supports smarter property investment decisions across London.

Rather than focusing solely on predicting house prices, the system integrates multiple urban datasets to evaluate investment opportunities based on affordability, accessibility, market value, and long-term growth potential.

The final outcome is an explainable investment recommendation engine that helps users identify promising London neighborhoods using data-driven insights.

---

## Business Problem

Property investors often face an overwhelming amount of market information but lack a structured way to evaluate investment opportunities objectively.

This project addresses that challenge by building a Decision Intelligence System capable of identifying London neighborhoods that provide the best balance between:

- Affordability
- Transport accessibility
- Market value
- Long-term investment potential

The objective is to support evidence-based investment decisions rather than simply predict property prices.

---

## Project Objectives

- Integrate multiple London open datasets
- Perform data cleaning and feature engineering
- Explore market trends through data analysis
- Develop machine learning models
- Build an Investment Score for each area
- Explain recommendations using interpretable analytics
